== Description ==

This will allow you to display exacly how many users total are on your blog.

For more information on the plugin visit: http://vasthtml.com/js/total-users-plugin/

== Installation ==

This section describes how to install the plugin and get it working.
 
1. Upload `total-comments` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

Please visit http://vasthtml.com/js/total-users-plugin/ for usage instructions.
